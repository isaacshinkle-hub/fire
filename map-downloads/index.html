Map Downloads

Downloadable map products for cockpit and field use.

Daily Aircraft Map (ForeFlight KMZ)

Same resources as the interactive map: ATGS/HLCO, Lead/ASM, air tankers, scoopers, tanker bases, and GACC boundary outlines. Transparent overlay for ForeFlight so nav charts stay visible underneath. Tanker base labels include air-to-air AM frequencies where available.

Download Latest Daily Aircraft Map KMZ

Updated daily around 10:00am MST with the interactive map. Replace/reimport the previous day file in ForeFlight after each morning update. This is the primary cockpit product.

National Dispatch Boundary Map

Seamless national dispatch boundary overlay for ForeFlight. Includes dispatch area polygons, dispatch center points with phone + A/G + A/A on the map labels, and yellow tanker bases with A/A frequencies on labels. Transparent base so IFR/VFR charts remain visible.
